# UI 设计文档

## 一、设计语言

| 要素 | 方案 |
|------|------|
| **主题** | 暗色主题（深棕+米色羊皮纸纹理），仿古代卷轴 |
| **字体** | 标题：思源宋体 / 正文：思源黑体 / 数字：等宽字体 |
| **配色** | 主色：#8B6914(金) / 背景：#1A1206(深棕) / 文字：#E8D5B7(羊皮白) / 强调：#C62828(朱砂) |
| **间距** | 8px 基础网格单位，组件间距 16px |
| **圆角** | 4px (微妙)，与古代硬直线条相衬 |
| **图标** | 自制簡化像素风格图标 |

### 5.1 LeftPanel — 政务菜单（Demo 已实现）

```
左侧面板(240px / w-60)：
  - 手风琴式折叠，默认全部收起
  - 折叠项（按实际代码顺序）：
    谍报（招募密探 + 训练女间谍 / 特工列表 / 派遣 / 反间 / 俘虏）
     计谋（三层折叠：L1 美人计·离间·假情报·空城 ✅ / L2 釜底抽薪·调虎离山·暗渡陈仓等 11 计 / L3 国策态势以逸待劳·远交近攻等 8 策·单选开关）
    家族（女眷 / 姻亲·子女待/已登场 / 婚配 / 在野跟随；无父辈·无独立子女 tab）
    人事（搜索/登用 + 任命三轨 AppointPanel + 美女库存赏赐）
    外交（各势力进贡/献美/点化女间谍/结盟按钮）
    君主（结束回合）
    己方城池（快捷列表，点击跳转地图）
  - 内政/军事操作在右侧 RightPanel，左侧不重复
```

### 5.2 MapCanvas — 地图渲染

```
Konva Canvas 分层渲染：

Layer 1 — 地形底图
  ├─ 中国地形轮廓(十三州)
  ├─ 河流/山脉/湖泊
  └─ 道路连线(城市间行军路径)

Layer 2 — 城市标记
  ├─ 105 个城市圆形标记
  ├─ 颜色 = 所属势力
  ├─ 大小 = 城市规模(人口)
  └─ 右侧附带标签(城市名+兵力数)

Layer 3 — 部队标记
  ├─ 行军中的部队(三角箭头方向)
  ├─ 驻守部队(旗帜图标)
  └─ 鼠标悬停 → 显示 Tooltip(统帅/兵力/目的地)

Layer 4 — UI 叠加
  ├─ 势力边界(粗略渲染)
  ├─ 选中城市高亮边框
  └─ 事件定位闪烁指示

交互：
  - 点击城市 → RightPanel 切换到 CityDetail
  - 右击城市 → 快速菜单(出征/运输/查看武将)
  - 缩放：滚轮 0.5× ~ 2.0×
  - 拖拽平移地图
```

### 6.1 战斗Canvas

```
Layer 1 — 网格
  ├─ 六角格背景(平/林/山/水/墙 颜色区分)
  └─ Z-order：高地 > 平地 > 水面

Layer 2 — 部队
  ├─ 每支部队一个小旗杆 + 图标
  ├─ 旗帜颜色 = 势力
  ├─ 选中部队 → 绿色六角格高亮
  └─ 移动范围 → 淡黄色六角格高亮

Layer 3 — 特效
  ├─ 火(闪烁红黄色)
  ├─ 水(波纹)
  ├─ 陷阱(灰色隐藏)
  └─ 落石(褐色)

交互：
  - 点击己方部队 → 选中 + 移动范围显示
  - 点击移动高亮格 → 移动
  - 点击可攻击的敌方 → 攻击确认
  - 右击部队 → 部队信息 mini 面板
```

### 6.2 BattleCommandBar

```
底部操作栏(高 60px)：
  [移动]     — 点击后进入移动模式, BFS高亮显示可移动格
  [攻击]     — 点击后进入攻击模式, 显示可攻击敌方(红色)
  [火计]    — Demo 已实现：选我军 → 火计 → 点敌军（气力≥30）
  [计策 ▼]  — 全量设计：水计/落石/伏兵/…（后置）
              灰色=不可用(气力不足/条件不满足)
  [单挑]     — 向相邻敌将发起单挑(灰色=不可用)
  [阵型 ▼]  — 下拉选择可用阵型(当前+切换消耗)
  [待机]     — 结束本队行动
  [全军撤退] — 所有部队撤退(防高/统高时成功率)
```

## 九、交互流程

### 9.1 新游戏 → 主界面

```
MainMenu [新游戏] → ScenarioSelect [选207剧本] [选刘备] [开始]
  → GameLayout：Server生成GameState → Client渲染地图+TopBar+LeftPanel+RightPanel
```

### 9.2 一个回合

```
1. 在 LeftPanel 逐项操作（开发/征兵/赏赐/出征）
2. 每次操作 → API调用 → Zustand store更新 → 地图重绘
3. 操作完成 → 点击"结束回合"
4. Server 处理 AI 回合 → 通过 WebSocket 逐势力推送变化
5. 全部完成 → 新回合开始
```

### 9.3 出征

```
LeftPanel → 军事 → 出征 → 弹出 BattleSetupModal
  → 选择出征城市 + 主将 + 副将
  → 选择参谋（智力≥85的同城武将列表；可选）
  → 选择兵种 + 阵型 + 兵数 + 粮草
  → 选择目标城市
  → [确定] → Army 出现在地图上，开始行军
```

### 9.4 战斗

```
两支敌对 Army 相遇 → Server 自动启动 Battle
  → Client 路由切换至 BattleView
  → 加载六角网格 + 双方部队
  → 每回合：移动 → 攻击/计策 → 待机
  → 胜负判定 → 切换回 GameLayout
```

### 9.5 内政面板（新增）

```
内政指令面板：
  [持续开发] — 选择维度(农/商/文/工/交/卫) → 指派武将 → 显示进度条
  [建造设施] — 设施槽(2~10格) → 选中建造 → 显示工期进度
  [城市政策] — 下拉选10种政策 → 即时生效(可每季切换)
  [贸易调配] — 地图上拖动物资(木/铁/马)到目标城市
```
