# 武将五维参考文档

> 本文档为三国时期武将的五维（统率/武力/智力/政治/魅力）建议值。
> 综合《三国志》《后汉书》《资治通鉴》等正史记载，
> 参考《三国演义》文学形象及历史文献记载，
> 适用于本游戏 0-B 全量数据录入时的属性定稿参考。

## 数值约定

| 等级 | 范围 | 含义 |
|:----:|:----:|------|
| 天花板 | 100（裸） | 每维仅 1 人；另有隐藏加成 |
| 超一流 | 96~99 | 顶尖水平，史上寥寥数人 |
| 一流 | 90~95 | 顶级人物 |
| 二流 | 80~89 | 优秀水准 |
| 三流 | 70~79 | 合格水准 |
| 中下 | 60~69 | 普通 |
| 低 | <60 | 薄弱 |

天花板体系见 `shared/ceiling.ts`：
- **统 100** 曹操（隐藏+15=115）
- **武 100** 吕布（隐藏+50=150）
- **智 100** 诸葛亮（隐藏+20=120）
- **政 100** 荀彧（隐藏+10=110）
- **魅 100** 刘备（隐藏+5=105）

第二梯度起点：武 97 / 其余 99。
