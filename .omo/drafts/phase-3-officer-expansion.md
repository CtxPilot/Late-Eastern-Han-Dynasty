## 数据填充规则

### 隐式字段默认值（参考已有武将 pattern）
- **compatibility**: 群雄=50，后期人物=50
- **righteousness**: 武将 8~12，文官 10~14
- **ambition**: 君主型 8~14，武将型 6~8，隐士型 2~4
- **valor**: 猛将 5~7，文官 1~3
- **composure**: 猛将 3~5，文官 5~7
- **lifespan**: 按实际死亡年填写
- **growth**: 核心人物 "mid"，边缘 "low"
- **personality**: 按演义/史实性格
- **ideal**: 群雄默认为 "hegemony"，后期按势力
- **bloodline**: 有血缘关系则填写，否则 []
- **ceilingBonus**: 非天花板人物 = null
- **power/burst/agility/luck**: 统武型偏高，文官偏低
- **intuition/awe**: 谋士偏高
- **strategy/tactics**: 按参考文档"谋略/奇谋"列

### unitProficiency
- 武将型：步兵 A/B，骑兵 B，水军 C
- 文官型：全 C 或 NONE（水军）
- 猛将型：对应兵种 S/A
- 南蛮型（孟获/祝融）：步兵 A，骑兵 C，水军 NONE

### formationMastery
- 武将：\[0, 2, 4, 7]（方阵/锋矢/鹤翼/鱼鳞）
- 文官：\[0]
- 名将：+ 专属阵型

### skills
- 核心人物 2~3 个技能，边缘人物 1~2 个
- 按人物特点选取（参考同类型已有武将）

### tags
- 按格式 `[籍贯, 社会出身, 职业, 特殊标签]`
